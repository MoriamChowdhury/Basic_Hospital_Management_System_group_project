# Basic_Hospital_Management_System_group_project
A console-based Hospital Management System in C using Code::Blocks,developed as a group project, managing patients, doctors, and appointments with basic billing.

Features:

Add, view, edit, and delete patients and doctors

Schedule, view, and cancel appointments

Generate basic bills

Save/load data from text files


Future Plans:

Full billing history, search/filter options, GUI, and role-based login


Technologies: C, Code::Blocks, Console Interface
